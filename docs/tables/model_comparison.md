# Model Type Comparison

| Model Type         | Main Goal           | Key Idea                               |
|--------------------|---------------------|----------------------------------------|
| **Dense**          | Raw capability      | Single large network                   |
| **MoE**            | Efficiency          | Specialists                            |
| **Agentic**        | Autonomy            | Act + plan                             |
| **RAG**            | Accuracy            | External memory                        |
| **Tool-use**       | Precision           | External skills                        |
| **Small models**   | Deployment          | Optimize learning                      |
| **Multimodal**     | Realism             | Multiple senses                        |
| **World models**   | Simulation          | Predict environments                   |
| **Multi-agent**    | Complexity          | Collaborative intelligence             |
| **Reasoning models**| Depth              | Structured thinking                    |
